# mean-stack-frontend

## Steps to run

Clone the Repo
``` 
git clone https://github.com/dj-mongodb/mean-stack-frontend
```

Change Directory
``` 
cd mean-stack-frontend 
```

Install Dependencies
```
npm install -g @angular/cli
```

Add Material UI Library
```
ng add @angular/material
``` 

Run Server
```
ng serve -o
```
